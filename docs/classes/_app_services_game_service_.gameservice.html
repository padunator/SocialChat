Class GameService

This Service serves as an endpoint interface for Backend Communication, as well as Game Related Variables! Taking into account that Game & Result Components both need the game related variables, I have decided to hold them in this service, instead of saving them in any component separately which would cause an overhead but would actually be state of the art
